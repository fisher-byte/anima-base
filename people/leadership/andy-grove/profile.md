# Andy Grove

**Andy Grove**（1936-2016）是英特尔联合创始人兼前CEO，被誉为硅谷管理教父，《High Output Management》作者，以其系统化、数据驱动的管理方法而闻名，定义了现代企业管理实践。

## 核心概念详解

### 1. Managerial Output
管理者的输出不是个人贡献，而是团队产出：
- **公式**: 管理者输出 = 组织产出中归因于管理者的部分
- **意义**: 管理者通过团队产生结果
- **应用**: 评估管理者看团队表现，不是个人任务

### 2. The Breakfast Factory Example
Andy用早餐工厂比喻管理：
- **早餐流程**: 采购 → 存储 → 准备 → 烹饪 → 服务
- **管理者角色**: 确保每个环节高效运行
- **关键**: 不是自己做早餐，是确保系统能做出好早餐

### 3. Performance Review
系统化绩效评估方法：
- **设定清晰期望** - 明确目标和标准
- **定期反馈** - 持续的正面和负面反馈
- **坦诚评估** - 准确反映表现
- **发展计划** - 改进和成长机会
- **文档记录** - 系统化记录和追踪

### 4. Meetings Management
有效会议管理：
- **会议前**: 明确目的和议程
- **会议中**: 控制时间和聚焦
- **会议后**: 文档化决策和行动项
- **会议频率**: 最小化必要会议
- **决策导向**: 每次会议应该有决策

### 5. Leverage Analysis
识别高杠杆活动：
- **高杠杆**: 影响最大，重复使用
- **低杠杆**: 影响小，一次性
- **管理者角色**: 识别并专注于高杠杆活动
- **示例**: 制定战略 > 处理日常运营
