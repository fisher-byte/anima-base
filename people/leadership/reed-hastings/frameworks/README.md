## No Rules Rules 核心三步曲

Reed Hastings 在《No Rules Rules》中提出的三个循序渐进的步骤：

```
Step 1: Talent Density（人才密度）
    ↓
Step 2: Candor（坦诚）
    ↓
Step 3: Freedom（自由）
```

**逻辑**：
- 只有先建立高人才密度，才能安全推行坦诚文化
- 只有在高人才密度 + 坦诚的基础上，才能给予自由

## 核心概念速查

| 概念 | 定义 |
|------|------|
| **Context, Not Control** | 给团队背景信息而非控制每个决定 |
| **Informed Captain** | 每个重大决策指定一个负责人 |
| **Highly Aligned, Loosely Coupled** | 目标对齐但执行独立 |
| **Sunshining** | 公开分享问题和错误 |
| **Disagree and Commit** | 可以争论，但决策后全力执行 |
| **4A Feedback** | Aim to Assist, Actionable, Appreciate, Accept/Discard |
| **Brilliant Jerks** | 能力出色但破坏文化的人，必须离开 |
