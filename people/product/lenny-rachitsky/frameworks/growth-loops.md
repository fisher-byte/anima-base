## 概述

Growth Loops 是可自我强化的增长系统，与传统漏斗不同，loops 的输出会成为新的输入，形成复利增长。Lenny Rachitsky 系统总结了最成功产品的 growth loops 模式。

## 核心概念

### Funnel vs Loop

**传统漏斗 (Funnel):**
```
投入 → 获客 → 激活 → 留存 → 变现
（线性，需要持续投入）
```

**增长闭环 (Loop):**
```
输入 → 行动 → 输出 
  ↑________________↓
（循环，自我强化）
```

**关键区别:**
- Funnel 需要持续燃料（广告预算）
- Loop 能自我驱动（用户带来新用户）

## 主要 Growth Loop 类型

### 1. Viral Loop（病毒式循环）
**机制:** 用户邀请更多用户

**公式:**
```
Viral Coefficient (K) = (每用户邀请数) × (邀请转化率)
K > 1 = 指数增长
K < 1 = 仍有价值，但需其他增长源
```

**案例: Dropbox**
```
用户使用 Dropbox
  → 分享文件夹给同事
  → 同事注册使用
  → 同事分享给更多人
  → 循环
```

### 2. Content Loop（内容循环）
**机制:** 用户创造内容吸引新用户

**案例: Pinterest**
```
用户 Pin 图片
  → 图片被 Google 索引
  → 搜索用户发现内容
  → 新用户注册并 Pin
  → 循环
```

### 3. Paid Loop（付费循环）
**机制:** 收入再投资于获客

**案例: Amazon**
```
销售商品
  → 获得利润
  → 投资广告
  → 获取更多用户
  → 更多销售
  → 循环
```

### 4. Marketplace Loop（市场循环）
**机制:** 供需双边互相促进

**案例: Uber**
```
更多乘客
  → 吸引更多司机
  → 更快响应时间
  → 吸引更多乘客
  → 循环
```

### 5. Retention Loop（留存循环）
**机制:** 使用越多价值越大

**案例: Notion**
```
用户创建笔记
  → 积累更多内容
  → 切换成本增加
  → 使用频率增加
  → 创造更多内容
  → 循环
```

## 设计 Growth Loop 的步骤

### Step 1: 识别核心行为
- 用户的核心动作是什么？
- 这个动作能带来什么输出？

### Step 2: 连接输出到输入
- 输出如何吸引新用户？
- 或如何让现有用户更活跃？

### Step 3: 优化循环速度
- 如何加快循环？
- 减少摩擦点

### Step 4: 测量循环健康度
- Cycle Time（循环时间）
- Conversion Rate（转化率）
- Output Quality（输出质量）

## 关键指标

### Loop Velocity（循环速度）
完成一个循环需要多长时间？
- 快速循环 = 快速增长
- 优化目标：减少循环时间

### Loop Strength（循环强度）
每次循环产生多少新输入？
- K 值（病毒系数）
- Retention rate
- Monetization efficiency

### Loop Compounding（复利效应）
多个循环如何互相增强？
