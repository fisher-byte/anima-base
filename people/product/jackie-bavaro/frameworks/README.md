## 框架概览

### Product Strategy Framework
**Vision + Strategic Framework + Roadmap**

产品策略的三个核心组成部分：
- Vision（愿景）：WHERE you want to go
- Strategic Framework（战略框架）：HOW you're going to get there
- Roadmap（路线图）：WHAT you're going to do

### PEARL Framework
**Problem → Epiphany → Action → Result → Learning**

替代传统 STAR 框架的行为面试回答方法：
- 强调洞察力（Epiphany）
- 强调学习成长（Learning）
- 专为高级 PM 设计

### Six Seniority Signals
1. **Autonomy / Agency** - 自主性
2. **Strategy** - 战略能力
3. **Tough Decision Making** - 困难决策
4. **Influencing Others** - 影响力
5. **Relevant Experience** - 相关经验
6. **Confident Grasp of Basics** - 基础能力掌握
