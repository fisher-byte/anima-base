## 跨播客反复出现的主题

### 1. 产品策略三要素
**Vision + Strategic Framework + Roadmap**
- 在多个播客中反复强调
- 是 Jackie 核心方法论之一

### 2. 避免 Feature Factory
- 主动制定策略而非被动接受需求
- 将功能与业务目标联系
- 用数据支持决策

### 3. 职业发展主动性
- 主动寻求更大责任
- 建立跨团队影响力
- 投资于沟通技能

### 4. 面试和职业准备
- PEARL 框架代替 STAR
- 展示资深信号
- 准备10个核心故事
