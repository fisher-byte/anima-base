# 2016微信公开课PRO·张小龙首次公开演讲

> **演讲时间**：2016年1月11日  
> **地点**：广州  
> **意义**：张小龙首次在微信公开课上公开演讲，系统阐述微信产品价值观

## 核心主题

### 1. 一切以用户价值为依归

**核心观点**：用户价值是第一位的，而非资源交换或商业合作

**关键论述**：
- "一切以用户价值为依归"是腾讯公司的核心价值观
- 真正把用户价值落实到产品的公司不多，大部分只是口头禅
- **不与任何人做资源交换来取代用户价值**
- 微信的挑战不是多做事情，而是能够挡掉多少不必要的事情

**实践案例**：
- **拒绝白名单特权**：系统没有超过5000好友的账号，"白名单是系统的瑕疵"
- **拒绝红包特权**：关闭了土豪红包白名单，避免用户攀比和不均衡
- **保护用户体验**：不会有系统群发消息，需要做很多事情才能让微信内容保持干净

### 2. 让创造发挥价值

**公众平台的本质**：让真正有价值的东西发挥出它的价值

**目标**：
- 提供平台让所有有才能的人都能触达用户
- 不限于互联网行业，而是各行各业
- 消除地理限制，让价值跨越物理边界

**比喻**："要让一个盲人在一个楼里给人按摩，他也能获得一个稳定的客户群"

### 3. 善良比聪明更重要

**引用亚马逊CEO**：《善良比聪明更重要》

**核心观点**：
- 欺骗用户是最容易做的事情，只需要聪明就可以
- 欺骗用户虽然容易获得流量和点击，但最终会把用户赶跑
- **怎么样对用户是好的，比聪明更重要**

### 4. 蝴蝶效应与平台责任

**2016年1月10日晚传播事件**：
- 微信公开课活动链接泄露 → 访问量暴增导致页面挂掉 → 造谣说会盗取支付宝账户 → 用户恐慌解绑银行卡

**教训**：
- 微信平台上，一点小疏漏会被放大无数次
- **微信作为基础信息传递平台，一个不经意的动作可能引起信息洪流的流向变化**
- 平台规则必须严格，因为影响量级很可怕

### 5. 应用号（小程序）预告

**首次透露**："我们尝试做到让更多的APP有一种更轻量的形态，但是又更好使用的一种形态来存在，这是我们在探讨的一种新的公众号形态，叫**应用号**。"

**时间节点**：这是小程序正式发布前一年的预告

## 产品哲学金句

1. "在一个移动互联网非常发达的时代，参加会议是挺浪费时间的"
2. "一切以用户价值为依归，用户价值是第一位的"
3. "白名单是一个系统的瑕疵"
4. "善良比聪明更重要"
5. "微信的挑战不是多做多少事情，而是能够挡掉多少事情"
6. "让真正有价值的东西发挥出它的价值"

## 历史意义

- **首次公开演讲**：张小龙首次在公开场合系统阐述微信价值观
- **确立价值观基石**：明确"用户价值第一"作为微信所有决策的底层逻辑
- **小程序预告**：首次透露"应用号"（后更名为小程序）的构想
- **反白名单文化**：明确反对特权和资源交换，坚持平台公平
