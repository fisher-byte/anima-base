## Teresa Torres 核心观点速览

### Opportunity Solution Tree (OST)
> 从业务成果出发 → 发现客户机会 → 生成解决方案 → 通过实验验证

### 持续发现的三个支柱
1. **Weekly Touchpoints**: 每周至少与一位客户对话
2. **Product Trio**: PM + 设计师 + 工程师共同发现
3. **Outcome over Output**: 关注成果而非功能

### 客户访谈原则
- 问故事，不问意见："Tell me about the last time..."
- 关注过去行为，不问未来意图
- 寻求真相，而非验证
