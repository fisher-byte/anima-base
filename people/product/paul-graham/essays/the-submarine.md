# The Submarine

> 原文链接: [paulgraham.com/submarine.html](https://paulgraham.com/submarine.html)

## 概述

Paul Graham 揭示了一个普通读者很少意识到的现实：**你在主流媒体上读到的很多"新闻"实际上是公关公司精心策划的结果**。这篇文章解释了 PR 行业如何像"潜水艇"一样在水面下操控媒体叙事。

## 核心观点

### 1. "潜水艇"比喻

- **水面之下**：PR 公司的工作是让新闻看起来像是自然发生的，而非人为推动
- **隐形操控**：好的 PR 不会让你意识到它的存在
- **读者是目标**：最终目标是影响你的认知，而你完全不知情

### 2. PR 如何运作

**记者的困境**：
- 记者需要持续产出内容
- 截止日期的压力让他们依赖现成的故事
- PR 公司提供"打包好"的故事，省去记者的调研工作

**PR 的策略**：
- 制造看起来像新闻的"趋势"（如"西装回归"）
- 通过多个渠道同时推送，制造"共识"假象
- 利用记者的懒惰和时间压力

### 3. 典型案例：西装"复兴"

Paul Graham 发现"西装回归"这个"趋势"在过去几年中被反复报道：
- 2005年4月、2005年2月
- 2004年9月、2004年6月、2004年3月
- 2003年9月、2002年11月...

**这不是巧合**——每次报道背后都是服装行业 PR 公司在推动。

### 4. 为什么这很重要

- **信息污染**：你以为是新闻的东西实际上是广告
- **认知操控**：你的"常识"可能是被人为塑造的
- **批判性思维缺失**：大多数人从不质疑他们读到的"趋势"

## 如何识别 PR 植入

### 危险信号
1. **趋势类报道**：任何声称"X 正在回归"或"Y 是新潮流"的文章
2. **匿名来源**：文章引用"业内人士"但没有具体姓名
3. **恰到好处的时机**：新闻恰好在产品发布或季度报告前出现
4. **缺乏反对声音**：只有正面观点，没有质疑

### 自我保护
1. **质疑"每个人都在做"的说法**：通常这只是 PR 话术
2. **寻找利益关联**：问自己"谁会从这个报道中获益？"
3. **交叉验证**：如果只在一两个来源看到，可能不是真正的趋势

## 金句摘录

> "One of the most surprising things I discovered during my brief business career was the existence of the PR industry."
> 
> 在我短暂的商业生涯中，我发现的最令人惊讶的事情之一是 PR 行业的存在。

> "The PR industry's goal is to make you think something is happening when it isn't."
> 
> PR 行业的目标是让你认为某件事正在发生，而实际上并没有。

## 对创业者的启示

1. **了解 PR 的运作**：作为创业者，你需要知道如何利用媒体
2. **但保持诚信**：不要制造虚假趋势
3. **真正的新闻更有价值**：如果你有真正有趣的故事，记者会主动来找你
4. **小心竞争对手的 PR**：他们的"势头"可能只是付费宣传
