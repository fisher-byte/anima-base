## 核心思想

### PG决策哲学
1. **第一性原理**
   - 从根本需求出发
   - 打破隐含假设
   - 重构解决方案

2. **反直觉思维**
   - 好想法看起来像坏想法
   - 质疑"不可能"
   - 证据 > 共识

3. **Relentlessly Resourceful**
   - 永不放弃
   - 找到方法
   - 创始人本质

4. **超线性回报**
   - 小差距 → 大结果
   - 复利效应
   - 长期主义
