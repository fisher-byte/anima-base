## 快速参考

### 5 组件速查

1. **Competitive Alternatives** — 客户的替代选择
2. **Unique Attributes** — 你的独特能力
3. **Value** — 客户获得的价值
4. **Target Market** — 最关心这些价值的客户
5. **Market Category** — 产品所属的市场类别

### 定位声明模板

```
For [目标客户]
Who [关键痛点/需求]
[产品名] is a [市场类别]
That [核心价值主张]
Unlike [主要替代方案]
We [关键差异化]
```
