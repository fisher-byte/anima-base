## 🧠 核心框架

### 财富创造

#### 1. [Specific Knowledge + Leverage](./frameworks/specific-knowledge-leverage.md)
**特定知识与杠杆：Naval的财富创造公式**

**核心公式：**
```
财富 = 特定知识 × 杠杆 × 责任感
```

**关键概念：**
- **特定知识**：不可被培训的、高度个人化的技能
- **四种杠杆**：劳动力、资本、代码、媒体
- **无需许可的杠杆**：代码和媒体（优先级最高）

**适用场景：**
- 职业规划和选择
- 创业方向决策
- 个人品牌建设
- 财富增长策略

---

#### 2. [Compound Interest Mindset](./frameworks/compound-interest-mindset.md)
**复利思维：Naval的人生增长模型**

**核心理念：**
> "All benefits in life come from compound interest—whether in money, relationships, love, health, activities, or habits."

**应用领域：**
- 💰 财富复利（投资、被动收入）
- 📚 知识复利（每日阅读、持续学习）
- 🤝 关系复利（深度友谊、长期合作）
- 💪 健康复利（每日锻炼、良好习惯）
- 🌟 声誉复利（诚信、长期建设）

**关键原则：**
1. 时间是最大的杠杆（早开始、保持连续）
2. 一致性胜过强度（每天进步1%）
3. 避免负复利（消除坏习惯）

---

#### 3. [Long-Term Thinking](./frameworks/long-term-thinking.md)
**长期思维：Naval的人生决策框架**

**核心原则：**
> "Play long-term games with long-term people."

**四大支柱：**
1. **与长期主义者合作**
   - 选择伙伴比选择机会重要
   - 如果不能合作20年，就不要合作1天

2. **建立长期声誉**
   - 声誉是最重要的资产
   - 用几十年建立，一天可以摧毁

3. **投资长期资产**
   - 技能和知识（永不贬值）
   - 关系和网络（信任需要时间）
   - 系统和平台（可扩展）

4. **做出长期决策**
   - 问："10年后我会后悔吗？"
   - 优先考虑长期声誉而非短期利益

### 幸福与智慧

#### 4. [Happiness as a Skill](./frameworks/happiness-as-skill.md)
**幸福是一种技能：Naval的幸福哲学**

**核心定义：**
```
幸福 = 内心平静 = 欲望的缺失
```

**三大原则：**
1. **减少欲望**（而非满足欲望）
2. **活在当下**（通过冥想训练）
3. **接受现实**（放下无法控制的）

**实践方法：**
- 🧘 每天冥想60分钟（"什么都不做"的冥想）
- 📝 审视欲望（区分真实需求和社会灌输）
- 🙏 感恩练习（专注已拥有的）
- 🌅 活在当下（减少内心独白）

**幸福习惯清单：**
- [ ] 每天冥想
- [ ] 充足睡眠（7-9小时）
- [ ] 规律运动（每天1小时）
- [ ] 健康饮食（少糖、少加工）
- [ ] 阅读经典（每天1-2小时）

---

#### 5. [First Principles Thinking](./frameworks/first-principles-thinking.md)
**第一性原理思维：Naval的清晰思考方法**

**核心方法：**
从最基本的真理出发，而非类比和模仿

**四个步骤：**
1. **识别假设**："这是真的吗？还是被灌输的？"
2. **分解问题**：拆解到最基本要素
3. **从基础重建**：寻找新的组合和解决方案
4. **验证和迭代**：实践检验

**应用案例：**
- 财富创造：特定知识 × 杠杆（而非上好大学、长时间工作）
- 幸福：减少欲望（而非满足欲望）
- 学习：基础学科 + 实践（而非追逐证书）

**常用心智模型：**
- 激励机制（Cui bono? 谁受益？）
- 复利思维
- 进化论（系统适应）
- 博弈论（零和 vs 非零和）
- 概率思维
