# Naval Ravikant - 核心框架索引

本目录包含Naval Ravikant的5个核心思维框架，涵盖财富创造和幸福哲学两大主题。

---

## 📋 框架列表

### 财富创造框架

#### 1. [Specific Knowledge + Leverage](./specific-knowledge-leverage.md)
**特定知识与杠杆：财富创造公式**

**核心公式：**
```
财富 = 特定知识 × 杠杆 × 责任感
```

**关键概念：**
- 特定知识：不可被培训的技能
- 四种杠杆：劳动力、资本、代码、媒体
- 无需许可的杠杆优先（代码和媒体）

**适用场景：**
- 职业规划
- 创业方向选择
- 个人品牌建设

**字数：** ~6,000字
**阅读时间：** 20分钟

---

#### 2. [Compound Interest Mindset](./compound-interest-mindset.md)
**复利思维：人生增长模型**

**核心理念：**
> "All benefits in life come from compound interest."

**应用领域：**
- 💰 财富复利
- 📚 知识复利
- 🤝 关系复利
- 💪 健康复利
- 🌟 声誉复利

**关键原则：**
1. 时间是最大杠杆
2. 一致性胜过强度
3. 避免负复利

**字数：** ~8,500字
**阅读时间：** 30分钟

---

#### 3. [Long-Term Thinking](./long-term-thinking.md)
**长期思维：人生决策框架**

**核心原则：**
> "Play long-term games with long-term people."

**四大支柱：**
1. 与长期主义者合作
2. 建立长期声誉
3. 投资长期资产
4. 做出长期决策

**决策工具：**
- 10-10-10 法则
- 10年后测试
- Cui bono 分析

**字数：** ~9,000字
**阅读时间：** 30分钟

---

### 幸福与智慧框架

#### 4. [Happiness as a Skill](./happiness-as-skill.md)
**幸福是一种技能：幸福哲学**

**核心定义：**
```
幸福 = 内心平静 = 欲望的缺失
```

**三大原则：**
1. 减少欲望（而非满足）
2. 活在当下（冥想训练）
3. 接受现实（放下控制）

**实践方法：**
- 🧘 每天冥想60分钟
- 📝 审视欲望
- 🙏 感恩练习
- 🌅 当下觉察

**字数：** ~7,500字
**阅读时间：** 25分钟

---

#### 5. [First Principles Thinking](./first-principles-thinking.md)
**第一性原理思维：清晰思考方法**

**核心方法：**
从最基本真理出发，而非类比

**四个步骤：**
1. 识别假设
2. 分解问题
3. 从基础重建
4. 验证迭代

**常用心智模型：**
- 激励机制
- 复利思维
- 进化论
- 博弈论
- 概率思维

**字数：** ~10,500字
**阅读时间：** 35分钟
