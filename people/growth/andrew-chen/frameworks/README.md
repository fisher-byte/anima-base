# Andrew Chen - Frameworks 框架索引

## 概览

Andrew Chen 作为增长领域的思想领袖，提出了多个影响深远的增长框架。他最著名的作品《The Cold Start Problem》系统化地总结了网络效应产品的启动和扩展方法论。

**收录数量**: 5 个框架  
**最后更新**: 2026-03-14

## 核心框架体系

### 🎯 冷启动问题（The Cold Start Problem）

**完整理论框架**：

1. [**Cold Start Problem 五阶段框架**](./cold-start-problem-five-stages.md)  
   网络效应产品从启动到成熟的完整路径  
   📌 五阶段: Cold Start → Tipping Point → Escape Velocity → Hitting the Ceiling → The Moat

2. [**原子网络理论（Atomic Network Theory）**](./atomic-network-theory.md)  
   解决冷启动问题的核心概念：最小可行网络  
   📌 关键案例: Tinder校园策略、Slack团队策略、信用卡Fresno策略

3. [**网络效应三大类型**](./three-network-effects.md)  
   Acquisition Effect（获客）+ Engagement Effect（参与）+ Economic Effect（经济）  
   📌 不同阶段的网络效应如何协同作用

## 实践工具包

### 📋 原子网络设计清单

使用 [原子网络理论](./atomic-network-theory.md) 时：

- [ ] 定义最小可行网络（最少需要多少用户？）
- [ ] 识别地理/社群边界
- [ ] 评估用户同质性
- [ ] 选择第一个原子网络
- [ ] 设计启动策略
- [ ] 设置成功指标
- [ ] 验证并总结手册
- [ ] 复制到下一个网络

### 📈 增长阶段诊断

使用 [五阶段框架](./cold-start-problem-five-stages.md) 时：

**我的产品处于哪个阶段？**

1. **Stage 1 (Cold Start)**: 
   - 用户 <1,000
   - 留存率低 (<20%)
   - 主要依靠创始人手动获客

2. **Stage 2 (Tipping Point)**:
   - 用户 1K-10K
   - 留存曲线开始扁平化
   - 有机增长开始出现

3. **Stage 3 (Escape Velocity)**:
   - 用户 10K-100K+
   - 快速扩张到多个市场
   - 病毒系数 >1

4. **Stage 4 (Hitting the Ceiling)**:
   - 用户 100K-1M+
   - 增长放缓
   - 反网络效应出现

5. **Stage 5 (The Moat)**:
   - 用户 1M+
   - 市场领导者
   - 建立竞争壁垒

### 🎯 增长渠道健康度检查

使用 [Law of Shitty Clickthroughs](./law-of-shitty-clickthroughs.md) 时：

对每个营销渠道评估：

- [ ] 当前 ROI 是多少？
- [ ] 过去 6 个月 ROI 趋势？
- [ ] 竞争对手是否已进入？
- [ ] 是否接近渠道生命周期末期？
- [ ] 有无新渠道候选？
