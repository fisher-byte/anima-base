# North Star Metric (NSM)

## 定义

**北极星指标(NSM)**是一个单一的、全公司范围的指标,用于对齐所有团队成员共同推动可持续增长。

> **"如果它对NSM不重要,你可能就不应该做这件事。"**
> 
> — Ed Baker (前Facebook国际增长负责人)

## 为什么NSM如此重要?

### Facebook案例
- **NSM**: 日活跃用户(Daily Active Users, DAUs)
- **执行方式**: 团队成员被鼓励"不断检查自己,将所做的一切与推动NSM联系起来"
- **目标设定**: 为NSM设定明确的6个月目标
- **结果**: 强大的团队对齐和可持续增长

### Uber案例
Ed Baker加入Uber后,第一件事就是与CEO Travis Kalanick确定NSM:
- **初步考虑**: 月活跃乘客数(Monthly Active Riders)
- **问题**: 作为双边市场,只反映增长的一部分
- **最终选择**: **每周乘车次数(Weekly Trips)**
- **原因**: 反映乘客和司机两侧的增长

## 如何选择NSM

### 第1步:理解客户如何获得价值

**关注点**:
- 不是任意客户,而是"必需品"客户
- 那些在[PMF Survey](./pmf-survey.md)中回答"非常失望"的用户
- 你的目标是在现有和新客户中扩展这种"必需品价值"

**前提条件**:
> 如果你还没有认为产品是"必需品"的客户,就不应该专注增长,而应该专注于[找到产品市场契合度](./pmf-survey.md)。

### 第2步:应该能够长期"向上向右"增长

**好的指标**:
- 可以在长期内持续增长
- 示例:日活跃用户(DAUs)、每周乘车次数

**不好的指标**:
- "30天留存率" — 最大值是100%,无法持续增长
- 有增长天花板的指标

### 第3步:考虑指标的负面影响

**思考场景**: 增长这个指标是否会导致团队行为违背公司长期利益?

**反面案例**:
- **"客户平均月收入"**
  - 问题:最快增长方式是淘汰低价值客户
  - 后果:即使这些客户有利可图,也会减少整体客户和收入增长
  - 违反:"向上向右"原则

- **"月度APP下载量"**
  - 问题:虚荣指标,可能驱动数百万次下载但从未使用
  - 更好选择:反映真实用户参与的指标,如"每周乘车次数"或DAUs

### 第4步:保持简单

**为什么简单很重要?**
- NSM的目的是对齐整个团队共同推动增长
- 必须足够简单,让每个人都能理解和记住
- 如果是复杂公式,团队成员会困惑并停止思考它

**常见问题**:
Sean在工作坊中发现:高管认为团队都知道NSM,但实际调查显示只有一小部分人知道。**几乎总是因为指标过于复杂**。

## 为什么不直接关注收入增长?

### 收入增长的局限
- 收入增长很重要,尤其对CEO而言(投资者和董事会的衡量标准)
- **挑战**:如果收入增长超过为客户提供的总价值增长,就不可持续

### 可持续增长逻辑
```
客户价值持续增长 → 可持续的收入增长
```

**最佳实践**:
- CEO追踪收入增长和NSM两个指标
- NSM作为可持续增长的驱动力
- 收入增长作为商业健康的结果指标

## NSM与公司使命的连接

### Facebook示例
- **NSM**: 日活跃用户(DAUs)
- **使命**: "让世界更紧密地联系在一起"
- **执行**: Zuckerberg在日常对话中不断强化这种连接

**威力来源**:
> 如果团队对使命有情感连接,并将NSM视为实现使命的记分卡,你将释放出可持续增长的强大力量。

## 实战指南

### NSM选择清单
- [ ] 反映"必需品"客户获得的核心价值
- [ ] 可以长期"向上向右"增长
- [ ] 不会引导团队做出违背长期利益的行为
- [ ] 简单到团队每个人都能理解和记住
- [ ] 与公司使命有明确连接

### 常见NSM示例

| 公司类型 | NSM示例 | 原因 |
|---------|---------|------|
| 社交网络 | 日活跃用户(DAUs) | 反映用户参与和网络价值 |
| 双边市场 | 每周交易次数 | 反映供需两侧健康 |
| SaaS产品 | 每周活跃团队数 | 反映产品粘性和价值 |
| 内容平台 | 每月内容消费时长 | 反映内容质量和用户参与 |

### 避免的陷阱
1. **过于复杂的公式** → 团队无法记住和执行
2. **虚荣指标**(如下载量) → 无法反映真实价值
3. **有增长天花板的指标**(如留存率) → 无法长期"向上向右"
4. **可能引导错误行为的指标**(如ARPU) → 损害长期增长

## 关键原则

> **"北极星指标可能是驱动可持续增长最强大的概念。"**
> 
> — Sean Ellis

**核心思想**:
- NSM是量化PMF价值扩展的方式
- 通过单一指标对齐整个组织
- 将日常工作与公司使命连接
- 驱动可持续而非短期增长
