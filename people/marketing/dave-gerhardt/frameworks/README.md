## 快速参考

### 营销核心法则
1. 没人想被营销
2. 不要等着才开始营销
3. 不要委员会式营销
4. 与买家建立信任
5. 放弃传统案例研究

### 对话式营销四步
1. Engage - 吸引
2. Understand - 理解
3. Recommend - 推荐
4. Close - 成交

### Founder Brand 三阶段
1. 掌握讲故事
2. 广泛分享
3. 从反馈中学习
